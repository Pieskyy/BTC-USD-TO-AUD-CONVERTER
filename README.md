# BTC-USD-TO-AUD-CONVERTER
Gets the current price of BTC and converts it to AUD.

Get your own keys you bums.

This uses the API from API Ninjas to get the current value of Bitcoin

https://api-ninjas.com/api/bitcoin

then it gets the exchange rates from exchangeratesapi

https://exchangeratesapi.io/

This change from EUR into AUD and USD so i have to get the EUR to USD then the EUR to AUD and do AUD / USD to get the price.

if you have any issues, slide me a message!
